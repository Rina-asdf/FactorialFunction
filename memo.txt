問題2：NodeとTypescriptを使って正整数Nを入力するとNの階乗を表示する関数を作ってください
- Nの階乗：
  - Nの階乗はN!と表します
  - たとえばN=4のとき 4! = 4 x 3 x 2 x 1 = 24となり
  　　　　　N=5のとき 5! = 5 x 4 x 3 x 2 x 1 = 120となります
- 要件：
  - 以下のような関数を定義すること
    - ex) const function = (N) => N!
  - 関数の引数と返り値の型を定義すること
  - 再帰関数を使うこと
  - 入力値を変更して関数を実行するごとに、正しく計算されたNの階乗の結果が何かしらの方法で表示・確認できること
